# Тесты для лабораторной работы №10
## Входные данные
На вход программе подается три ключа: Входной файл, выходной файл, ключ действия
Файл содержит информацию о товаре: название, категория, количество, цена.
По ключу missing программа возвращает отсутствующие товары
По ключу sort программа сортирует товары по их возможной выручке в порядке возрастания
По ключу sortrev программа сортирует товары по их возможной выручке в порядке убывания
По ключу insert программа добавляет товары из второго файла в первый.

## Выходные данные
Сумма матриц, произведение матрицы или определитель матрицы
## Позитивные тесты:
- 01 - missing - такие продукты есть
- 02 - missing - таких продуктов нет
- 03 - sort - сортировка файла из одного товара
- 04 - sort - сортировка неотсортированного файла с несколькими товарами
- 05 - sortrev - сортировка файла из одного товара
- 06 - sortrev - сортировка неотсортированного файла с несколькими товарами
- 07 - insert - первый файл пустой
- 08 - insert - второй файл пустой
- 09 - insert - Оба файла непустые


## Негативные тесты:
- 01 - missing - Пустой файл
- 02 - Файл отсутствует
- 03 - Ключ действия неверный
- 04 - Файл содержит букву в количестве товара
- 05 - Отрицательное количество
- 06 - Неположительная цена
